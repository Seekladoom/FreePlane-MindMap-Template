# Freeplane 导出文件格式 备忘录
PNG的文本不可复制（废话）

PDF并不是所有字体（比如思源黑体、更纱黑体，不转曲的话，在导出时字形就会崩）都能导出字形正常的思维导图，因此需要转曲确保在不管用什么字体的情况下都能导出字形正常的思维导图。

在Windows系统下，如果要规避字体版权又想直接用系统自带字体确保他人也能正常显示的话，可以直接用黑体，即simhei，也就是中易黑体。
